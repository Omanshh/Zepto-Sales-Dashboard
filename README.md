# Zepto-Sales-Dashboard# Zepto Sales Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes Zepto's sales performance across different outlet types, sizes, and product categories.

## Tools Used
- Power BI
- DAX
- Power Query
- Excel

## Key Insights
- Tier 3 outlets generated the highest sales.
- Low-fat products contributed the majority of revenue.
- Sales peaked in 2023 and 2024.
